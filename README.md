This is a minimal Streamlit web app to demo speech emotion recognition (4 classes).
It is prepared for quick deployment to Streamlit Cloud or HuggingFace Spaces.

## Quick start (local)
```bash
python -m venv venv
venv\Scripts\activate   # Windows
pip install -r requirements.txt
streamlit run app.py
```

## Deploy to Streamlit Cloud
1. Push this repository to GitHub.
2. On https://share.streamlit.io create a new app pointing to this repo and `app.py`.
